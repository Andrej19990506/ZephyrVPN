# Парсер платформы Гуляш

Скрипт для анализа структуры и функционала платформы "Гуляш" с целью выявления полезных и лишних элементов.

## Установка

```bash
cd scripts
npm install
```

## Использование

1. **Отредактируйте конфигурацию** в `parse_gulyash_platform.js`:
   ```javascript
   const CONFIG = {
     baseUrl: 'https://gulyash.ru', // Замените на реальный URL
     outputDir: './parsed_data',
     screenshots: true,
     waitTime: 3000,
   };
   ```

2. **Запустите парсинг**:
   ```bash
   npm run parse
   # или
   node parse_gulyash_platform.js
   ```

## Что делает скрипт

1. **Открывает главную страницу** и парсит:
   - Заголовки (h1-h6)
   - Кнопки и ссылки
   - Формы
   - Навигацию

2. **Находит все разделы** сайта через навигацию

3. **Парсит каждый раздел**:
   - Таблицы и их структуру
   - UI элементы (карточки, панели, виджеты)
   - Модальные окна
   - Формы

4. **Анализирует данные**:
   - Ищет полезные функции (заказы, меню, касса, кухня, персонал, отчеты и т.д.)
   - Ищет лишние функции (реклама, блог, новости, социальные сети и т.д.)

5. **Сохраняет результаты**:
   - `full_data.json` - полные данные в JSON
   - `report.md` - отчет в Markdown
   - Скриншоты всех страниц (если включено)

## Результаты

После выполнения в папке `parsed_data` будут:

- `full_data.json` - все собранные данные
- `report.md` - структурированный отчет
- `*.png` - скриншоты страниц

## Настройка ключевых слов

В функции `analyzeData()` можно добавить свои ключевые слова:

```javascript
const usefulKeywords = [
  'заказ', 'order', 'меню', 'menu', // ... добавьте свои
];

const unnecessaryKeywords = [
  'реклама', 'ad', 'banner', // ... добавьте свои
];
```

## Примечания

- Скрипт работает в видимом браузере (headless: false) для отладки
- Можно изменить на `headless: true` для фонового режима
- Увеличьте `waitTime` если страницы загружаются медленно
- Скрипт автоматически скроллит страницы для загрузки динамического контента
